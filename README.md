# Mobile-Applicaiton-Development (CSE-414)

🚀 Overview

Mobile applications are essential in modern digital systems.
This repository is dedicated to developing responsive, scalable, and efficient mobile applications while practicing Flutter development and mobile UI design.

🛠️ Tech Stack

Flutter

Dart

VS Code

Android SDK

Git & GitHub

⚙️ Environment Setup

Follow these steps to configure the development environment.

1️⃣ Install Flutter

Download the Flutter SDK:

https://flutter.dev/docs/get-started/install

Extract the SDK and add Flutter to the system path.

Verify installation:

flutter doctor
2️⃣ Install VS Code Extensions

Open VS Code → Extensions (Ctrl + Shift + X) and install:

Flutter

Dart

These extensions provide Flutter debugging, code completion, and project templates.

3️⃣ Verify Flutter Setup

Run:

flutter doctor

This command checks whether Flutter and required tools are installed correctly.

▶️ Running the Project

Clone the repository:

git clone https://github.com/your-username/mobile-app-development.git

Navigate to the project directory:

cd mobile-app-development

Install dependencies:

flutter pub get

Run the application:

flutter run
